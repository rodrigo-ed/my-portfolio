# **App Name**: Code Canvas

## Core Features:

- Homepage: Home page with a brief introduction and key skills.
- Project Gallery: Interactive project gallery showcasing past works with descriptions and links.
- About Me: Dedicated 'About Me' section with detailed background, skills, and personal interests.
- Contact Section: Contact form or direct email link for inquiries.
- Blog: Blog section featuring technical articles, personal insights, and industry thoughts.
- Skills Showcase: Skills section with proficiency levels visualized (e.g., using bars or circles).

## Style Guidelines:

- Primary color: Dark blue (#34495E) to evoke professionalism and stability.
- Background color: Very light blue (#F0F4F8) to create a clean and modern backdrop.
- Accent color: Purple (#9B59B6) to add a touch of creativity and stand out in calls to action.
- Body text: 'Inter' sans-serif for readability and modern feel. Headline Text: 'Space Grotesk' for a modern touch.
- Simple, outlined icons for navigation and skills representation.
- Grid-based layout for responsive design across devices.
- Subtle animations on page load and when hovering over project cards.
